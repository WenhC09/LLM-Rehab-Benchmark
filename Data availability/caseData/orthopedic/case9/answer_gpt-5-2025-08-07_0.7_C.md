# 个性化康复治疗计划方案

## 1. 康复目标

### 短期目标（2-4周）
1. 缓解左踝足疼痛（VAS ≤ 3分），减轻局部肿胀（“8”字形围度减少 ≥ 2 cm）。
2. 改善左踝关节活动度：背伸 ≥ 0°，跖屈 ≥ 30°。
3. 提升左下肢肌力：比目鱼肌、腓肠肌AEMG提高 ≥ 15%。
4. 提升站立平衡至3级，改善静态平衡重心分布。
5. 步行功能提升至Holden 3级，可在辅助用具下平地行走 ≥ 50 m。
6. 改善ADL评分至 ≥ 75分，提升如厕与转移的独立性。

### 长期目标（8-12周）
1. 左踝关节活动度接近健侧：背伸 ≥ 10°，跖屈 ≥ 40°，内外翻恢复 ≥ 10°。
2. 左下肢肌力恢复至接近健侧（AEMG差异 ≤ 10%）。
3. 恢复正常步态，可无辅助用具平地行走 ≥ 500 m，上下楼梯独立完成。
4. 平衡功能恢复至4-5级，动态平衡可独立完成日常活动。
5. ADL评分 ≥ 95分，生活完全自理。
6. 社会参与度明显提高（ICF评分 ≥ 90分），重返正常工作及社会活动。

---

## 2. 治疗方法

### 2.1 疼痛与肿胀管理
- **物理因子治疗**：
  - 低频脉冲电刺激（TENS）：每日1次，20分钟，VAS ≥ 3时使用。
  - 中频干扰电疗：促进血液循环，减轻水肿。
  - 冷疗：活动训练后冰敷15分钟，减少炎症反应。
- **水肿消肿**：
  - 伤肢抬高（高于心脏平面），每日3-4次，每次20分钟。
  - 弹力绷带或弹力袜分阶段加压。

### 2.2 关节活动度训练
- **被动及主动辅助活动**（注意保护跟腱修复部位）：
  - 早期（术后6周内）：踝关节背伸、跖屈的被动牵伸，幅度在疼痛可耐范围内。
  - 中期（6-8周）：逐步加入主动牵伸和抗阻训练，内外翻轻柔牵伸。
  - 每日2-3次，每次10-15分钟。
- **松动术**：
  - 踝关节关节松动技术（I-II级）改善关节囊柔韧性，防止粘连。

### 2.3 肌力训练
- **早期（保护期）**：
  - 等长收缩训练：比目鱼肌、腓肠肌、胫前肌，3组×10次，保持5秒。
  - 足趾屈伸、抓毛巾训练，促进小肌群力量恢复。
- **中期（负重期）**：
  - 弹力带抗阻训练，循序渐进增加阻力。
  - 跟提起（双脚→单脚），注意疼痛和肿胀反应。
- **晚期（功能期）**：
  - 间歇性跳跃、快走、变向训练，为运动或工作需求做准备。

### 2.4 平衡功能训练
- **静态平衡**：
  - 双脚站立重心转移训练（镜前反馈）。
  - 健足支撑→患足支撑过渡。
- **动态平衡**：
  - 平衡垫、半球训练板训练。
  - 交替抬腿、跨步、侧移步训练。
- **结合步态**：
  - 在平地、不同材质地面进行平衡与步行训练。

### 2.5 步行功能训练
- **早期**：
  - 在支具保护下，双拐或单拐步行训练，注意步幅对称。
- **中期**：
  - 渐进性部分负重训练（跑台或步态训练器），监测左足负重比例。
- **晚期**：
  - 自然步态训练，重点纠正支撑相伸髋、伸膝及踝背屈不足。
  - 上下楼梯训练。

### 2.6 日常生活能力与社会参与训练
- **ADL训练**：
  - 针对如厕、转移、上下楼、洗澡等进行情境模拟训练。
- **职业康复**：
  - 根据患者工作性质，安排专项功能训练，恢复劳动技能。
- **心理支持与教育**：
  - 解释康复进展，增强康复信心，指导自我管理和家庭康复训练方法。

### 2.7 康复教育与家庭训练
- 教授自我牵伸和肌力训练方法。
- 监测疼痛、肿胀变化，出现异常及时就医。
- 强调康复过程中的循序渐进，避免过早高强度负荷。

---

**注意事项**：
- 严格遵循跟腱修复术后康复保护原则，避免过度牵拉和负重，以防再断裂。
- 每周评估疼痛、肿胀、活动度及肌力变化，及时调整康复计划。

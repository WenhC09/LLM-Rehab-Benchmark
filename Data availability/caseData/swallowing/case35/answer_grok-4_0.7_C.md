# 康复治疗计划方案

## 1. 康复目标

### 短期目标（1-3个月）
- 改善吞咽安全性，减少隐性误吸，PAS分级降至6级以下。
- 提高咳嗽反射和气道廓清能力，SCSS评分提升至5分。
- 提高肺功能，FVC提升至2.0L以上，MEP提升至50cmH₂O以上。
- 改善营养状态，体重增加3-5kg，NRS-2002评分降至2分以下。
- 提高ADL评分至70分，减少完全依赖状态。
- 维持血氧饱和度在无吸氧状态下≥92%。

### 长期目标（3-12个月）
- 逐步过渡至安全经口进食，拔除鼻胃管，V-VST测试通过3号食物10mL。
- 恢复咳嗽反射和自主清嗓能力，SCSS评分达7分以上。
- 改善肺功能至接近正常水平（FVC≥2.5L，MEP≥60cm₂O）。
- 恢复舌骨上肌群力量，sEMG波幅恢复至正常范围。
- 提高ADL评分至85分以上，部分自理。
- 减少吸入性肺炎复发，CRP降至正常范围，体温稳定。

## 2. 治疗方法

### 2.1 吞咽功能康复
- **吞咽姿势训练**：
  - 采用坐位或半卧位，保持头部前倾、下颌内收姿势，减少误吸风险。
  - 指导左转头和右转头吞咽，每次训练10次/组，3组/天，持续4周。
  - 每周评估吞咽姿势对残留清除效果的影响，调整姿势角度。

- **舌骨上肌群强化训练**：
  - **Mendelsohn手法**：指导患者在吞咽过程中延长舌骨上提时间，每次保持3-5秒，10次/组，3组/天，持续6周。
  - **Shaker训练**：仰卧位下抬高头部并保持10秒，10次/组，2组/天，4周后评估舌骨上肌群sEMG波幅变化。
  - **阻力训练**：使用阻力带进行舌骨上肌群抗阻运动，5次/组，2组/天，强度逐渐增加，持续8周。

- **吞咽功能刺激**：
  - **冰刺激**：在咽部及会厌区域进行冰刺激，每次5分钟，2次/天，刺激咽反射，持续4周。
  - **电刺激（NMES）**：针对舌骨上肌群及咽部肌肉，使用低频电刺激（频率20-50Hz，强度以患者耐受为限），每次20分钟，5次/周，持续6周。
  - **生物反馈训练**：结合sEMG反馈，指导患者感知舌骨上肌群收缩，10分钟/次，3次/周，持续8周。

- **饮食管理**：
  - 继续鼻饲营养，采用高蛋白、高热量配方（每日热量25-30kcal/kg，蛋白质1.5g/kg）。
  - 在吞咽安全评估（FEES或VFSS）显示PAS≤6级后，尝试经口进食高稠度食物（3mL/次），每周评估安全性。
  - 逐步增加进食量（5mL、10mL），根据V-VST结果调整食物稠度。

### 2.2 呼吸功能康复
- **呼吸肌训练**：
  - **膈肌训练**：使用呼吸阻力训练器（阈值设定为MEP的40%-60%），每次15分钟，2次/天，持续8周，逐步增加阻力。
  - **深呼吸训练**：指导患者进行腹式呼吸，每次10次/组，3组/天，持续6周，配合膈肌超声评估移动度变化。
  - **胸廓扩张训练**：指导患者进行胸式呼吸及胸廓扩张运动，每次10次/组，2组/天，持续6周，改善胸腔活动度。

- **咳嗽功能训练**：
  - **主动咳嗽训练**：指导患者在吸气后快速用力咳嗽，每次10次/组，3组/天，持续6周。
  - **辅助咳嗽**：由治疗师或家属辅助胸腹部挤压，增强咳嗽力量，每次5次/组，2组/天，持续4周。
  - **气道廓清技术**：使用振动排痰仪（频率15-20Hz），每次15分钟，2次/天，持续4周，配合体位引流（头低位，左侧卧位优先）。

- **气管切开管理**：
  - 继续佩戴说话瓣膜，逐步延长佩戴时间至60分钟/次，3次/天，观察血氧饱和度和气促情况。
  - 每周评估气囊抽空耐受时间，计划在呼吸功能稳定（FVC≥2.0L）后尝试拔除气管切开套管。

### 2.3 肺功能及全身耐力训练
- **有氧训练**：
  - 使用上肢功率自行车，初始强度为最大心率的50%，每次10分钟，3次/周，持续6周，逐步增加至15分钟。
  - 在血氧饱和度稳定（≥95%）后，尝试下床步行训练，初始距离50米，逐步增加至200米，3次/周。

- **抗感染管理**：
  - 配合抗生素治疗吸入性肺炎，监测CRP及白细胞水平，每周复查血常规。
  - 保持口腔卫生，使用改良口腔护理（MBOAS评分目标降至6分），每日2次氯己定漱口。

### 2.4 营养支持
- **营养干预**：
  - 制定鼻饲营养计划：每日热量1800-2000kcal，蛋白质80-100g，脂肪占总热量的30%，分6次鼻饲。
  - 每周监测体重及营养指标（蛋白质、albumin），目标体重增加0.5kg/周。
  - 补充微量元素（锌、维生素C）以促进免疫功能恢复。

- **营养教育**：
  - 指导家属选择适合的高稠度食物（如米糊、果泥），为经口进食做准备。
  - 提供营养风险管理教育，强调误吸预防。

### 2.5 ADL及功能训练
- **ADL训练**：
  - 指导患者进行床上翻身、坐起训练，每次10分钟，2次/天，持续6周，目标ADL评分提升至65分。
  - 逐步引入穿衣、洗漱等自理活动，5次/周，观察耐力及跌倒风险（MORSE评分目标降至25分）。

- **全身肌力训练**：
  - 针对四肢肌力，进行低强度抗阻训练（1kg哑铃或弹力带），每次10次/组，2组/天，持续6周。
  - 重点加强上肢及核心肌群，为ADL功能恢复提供支持。

### 2.6 心理及社会支持
- **心理干预**：
  - 每周1次心理咨询，缓解患者因长期卧床及吞咽困难引起的焦虑情绪。
  - 使用音乐疗法或放松训练，每次20分钟，2次/周，持续8周。

- **家属教育**：
  - 培训家属协助吞咽姿势、排痰及ADL训练，每周1次指导，持续12周。
  - 提供误吸预防及气管切开护理知识，确保家属掌握正确操作。

### 2.7 康复监测与调整
- **每周评估**：
  - FEES或VFSS评估吞咽功能，监测PAS分级及残留情况。
  - 肺功能检查（FVC、MEP），评估呼吸肌力量及胸腔活动度。
  - 血气分析及血常规，监测氧合状态及感染情况。

- **每月调整**：
  - 根据吞咽功能进展调整电刺激强度及饮食稠度。
  - 根据肺功能改善情况调整呼吸训练强度及气管切开管理计划。
  - 根据ADL评分及体重变化调整训练强度及营养计划。
